# Real-Time Robotic Perception and Multi-Agent Tracking

## Overview

This project focuses on the development of real-time computer vision systems for autonomous robotic combat applications.

The objective was to build low-latency perception pipelines capable of detecting, tracking, and localizing multiple robots in dynamic environments while operating under strict computational and real-time constraints.

---

## Problem Statement

Autonomous robotic systems require robust perception capabilities to:

* Detect targets in real time
* Track multiple agents simultaneously
* Maintain situational awareness
* Operate under limited computational resources
* Minimize inference latency

This project investigates the effectiveness of modern computer vision techniques for real-time robotic perception.

---

## System Components

### Object Detection

Implemented and evaluated:

* YOLO-based object detection models

Applications:

* Robot detection
* Target localization
* Environmental perception

---

### Object Tracking

Developed tracking pipelines using:

* MOG2 Background Subtraction
* LOBO Tracking
* Segment Anything Model 2 (SAM2)

The tracking system was designed to maintain stable object trajectories under dynamic and resource-constrained conditions.

---

### Multi-Agent Tracking

Built a top-down tracking framework capable of:

* Tracking multiple robots simultaneously
* Real-time localization
* Movement analysis
* Situational awareness

The system supports competitive robotic environments where multiple agents interact continuously.

---

## Engineering Challenges

Key challenges addressed:

* Low-latency inference
